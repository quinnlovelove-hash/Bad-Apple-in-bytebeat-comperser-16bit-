How to Use (Updated for Zopium)

    Get the Code:

        Run the Python script to generate the Hex data.

        Open the file with EmEditor to copy the massive code string.

    Setup Zopium:

        Go to zopium.neocities.org.

        Click Stop before pasting to ensure stability.

        Paste the code into the editor.

    Audio Configuration:

        Change the Sample Rate to 96000.

        Ensure the mode is set to Floatbeat (or staged if using the latest script logic).
